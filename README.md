# WFSTDM

NICTの前中期計画にて開発していた有限重み付きトランスデューサー型の対話制御システム(WFSTDM)と，
ブラウザ上でWFSTDMを用いた対話システムを構築するDMBuilder，および必要ファイルを公開します．

なお，これらの研究は前中期計画で終了し，現在は取り組んでおりませんので，

・as-is：一切のバージョンアップ等はありません

・non-support：サポート等は一切行いません

・無保証：このリポジトリによる不具合等は一切保証いたしません

・一切の質問はお受けいたしかねます

をご理解の上，自己責任にてご参照下さい．


wfstdm-2.12.0.tar.bz2およびcooktest-1.0.1.tar.bz2はxaaから.xagは，ダウンロードした後，catコマンドにて連結させてご利用下さい．

cat wfstdm-2.12.0.tar.bz2.xa* > wfstdm-2.12.0.tar.bz2

cat cooktest-1.0.1.tar.bz2.xa* > cooktest-1.0.1.tar.bz2


システム構築については「音声対話システム環境構築手順書」にしたがって構築できます．OSはCentOS 6を想定
しております．仮想環境でも構いません．その他の説明については他のPDFファイルを参照して下さい．
